# Skill Gap Template

If you encounter an Xsolla workflow that is not covered by an existing skill, file an issue using this template.

## Issue title

`skill-gap: <short description of missing workflow>`

## Issue body

```
**Missing workflow:**
[Describe what the agent should be able to do but cannot]

**Trigger prompt:**
[Exact prompt that produced wrong/incomplete results]

**Expected behavior:**
[What the agent should have done]

**Actual behavior:**
[What the agent actually did]

**Suggested skill name:**
[e.g. promotions-setup, analytics-config]

**Suggested owner:**
[Who at Xsolla owns this surface]
```
